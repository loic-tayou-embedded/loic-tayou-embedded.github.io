Chaîne UART – Servo – Télémètre

Type : Projet personnel – Carte DE1 Cyclone II – VHDL (tout hardware)

VHDL UART RX/TX PWM servomoteur Télémètre ultrason ModelSim Testbenches & scripts .do

Résumé du projet

Projet personnel inspiré d’un sujet de mini-projet universitaire : tous les blocs sont décrits en VHDL et la carte utilisée est la DE1 (pas de processeur, pas de Qsys). L’objectif est de construire une chaîne complète autour d’un FPGA permettant :

  • d’échanger des données avec un PC via une liaison série UART,
  • de piloter un servomoteur en PWM à partir des commandes reçues,
  • de mesurer une distance avec un télémètre à ultrasons et de renvoyer la mesure au PC.

Le système complet implémente un automate qui sélectionne le comportement en fonction des interrupteurs SW1:SW0 (Système UART/Servo/Télémètre) :

  • SW1 SW0 = 00 : mode idle, aucun IP (UART, servo, télémètre) n’est actif.
  • SW1 SW0 = 01 : mode UART + servo : le PC envoie une position via le terminal série (Putty), traduite en largeur d’impulsion PWM pour positionner le servomoteur.
  • SW1 SW0 = 10 : mode UART + télémètre : le FPGA commande le télémètre, récupère la distance mesurée et renvoie cycliquement la valeur vers le terminal PC.
  • SW1 SW0 = 11 : mode servo + télémètre synchronisés : le servo balaie l’environnement tandis que le télémètre mesure la distance, ce qui permet d’associer une mesure de distance à un angle donné.

Ce que j’ai réalisé

  • Implémentation VHDL des blocs UART RX/TX paramétrables (baudrate, format 8N1).
  • Générateur de PWM pour servomoteur (période = 20 ms, position contrôlée par registre).
  • IP télémètre ultrason (génération du pulse de déclenchement, mesure du temps d’écho, conversion en distance).
  • Écriture de testbenches dédiés (UART_TX_tb, UART_RX_tb, SERVO_tb, TÉLÉMÈTRE_tb) et scripts .do ModelSim pour automatiser les simulations.
  • Automate VHDL de supervision implémentant les 4 modes (idle, UART+servo, UART+télémètre, servo+télémètre).
  • Intégration de tous les blocs dans une top-level et validation du fonctionnement complet sur la carte DE1.

Techno & outils

FPGA DE1 Cyclone II, VHDL, UART, PWM, télémètre à ultrasons, Intel Quartus, ModelSim (testbenches + scripts .do).
